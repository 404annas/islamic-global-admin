SETUP INSTRUCTIONS (5 MINUTES)
================================================================================

1. Navigate to directory:
   cd admin-dashboard

2. Install dependencies:
   npm install

3. Create environment file:
   cp .env.example .env
   # Configure VITE_API_URL if needed

4. Start development server:
   npm run dev
   # Opens at http://localhost:5174

5. Login with admin credentials:
   Email: admin@islamic-global.com
   Password: [your admin password]

Detailed instructions in QUICKSTART.md

API ENDPOINTS (13 TOTAL)
================================================================================

Authentication (2):
  POST /api/admin/login          - Admin login
  POST /api/admin/logout         - Admin logout

User Management (5):
  GET    /api/admin/users                 - List users (paginated)
  GET    /api/admin/users/:userId         - Get user details
  PATCH  /api/admin/users/:userId/status  - Update status
  PATCH  /api/admin/users/:userId/block   - Block/unblock
  PATCH  /api/admin/users/:userId/payment - Update payment

Progress (2):
  GET    /api/admin/users/:userId/progress   - Get progress
  POST   /api/admin/users/:userId/progress   - Update progress

Analytics (2):
  GET /api/admin/analytics         - Get platform analytics
  GET /api/admin/dashboard/stats   - Get dashboard stats

All endpoints fully documented in API_DOCUMENTATION.md
